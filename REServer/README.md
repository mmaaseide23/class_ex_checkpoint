User stories:

As a homeowner, I want to list my house for sale, so I can sell my house.
As a purchaser, I want to create an account, so I can view houses.
As a purchaser, I want to be able to express interest in postal codes, so I can tailor my feed to what I want to see.

Next steps:

As a homeowner, I would like to remove my house listing, so I can take my house off the market.
As a purchaser, I would like to put in an offer, so I can try to buy a house.
As a homeowner, I would like to see properties near mine, so I can understand what other houses near me are selling for.
